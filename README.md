# 📊 UseCalcs - Percentage & Finance Suite

**UseCalcs** is a collection of 10 lightweight, interconnected web calculators designed to solve everyday mathematical problems, project quoting, and financial calculations quickly and effortlessly.

## 🚀 Included Calculators

* **Freelance Project Quoter:** Calculates the base cost of a project based on an hourly rate and adds an adjustable contingency buffer.
* **Markup vs Margin:** Clearly compares net profit margin versus markup percentage.
* **Expense Splitter:** Equitably divides group expenses for trips, events, or shared bills.
* **Discount & Sales Tax:** Calculates the final price of a product by applying discounts and adding state taxes.
* **Tip Calculator:** Calculates exact tips and splits the total bill among multiple people.
* **Commission Calculator:** Tool to calculate net earnings from sales.
* **Basic Calculations:** Percentage variations (increase/decrease) and finding the exact percentage of a given number.

## 🛠️ Technologies Used

* **HTML5 & CSS3:** Mobile-first architecture, ensuring an excellent experience on smartphones and responsive design.
* **JavaScript (Vanilla):** Real-time calculation logic and client-side data validation without heavy frameworks.
* **SEO & Analytics:** Implementation of structured meta tags, Google Site Verification, and traffic metrics tracking via Google Analytics (gtag.js).
