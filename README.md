# 📊 UseCalcs - Suite de Porcentajes y Finanzas

**UseCalcs** es una colección de 10 calculadoras web ligeras e interconectadas, diseñadas para resolver problemas matemáticos cotidianos, presupuestos de proyectos y cálculos financieros de manera rápida y sin complicaciones.

## 🚀 Calculadoras Incluidas

* **Freelance Project Quoter:** Presupuestador de proyectos. Calcula el costo base según el valor por hora y añade un margen de contingencia ajustable.
* **Markup vs Margin:** Compara de forma clara el margen de beneficio neto frente al porcentaje de sobreprecio.
* **Expense Splitter:** División equitativa de gastos grupales para viajes o eventos.
* **Discount & Sales Tax:** Calcula el precio final de un producto aplicando descuentos y sumando impuestos estatales.
* **Tip Calculator:** Calcula propinas exactas y divide el costo total de la cuenta entre varios comensales.
* **Commission Calculator:** Herramienta para calcular ganancias netas por ventas.
* **Cálculos Básicos:** Variaciones porcentuales (incremento/decremento) y cálculo del porcentaje exacto de un número.

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Arquitectura basada en Mobile First, garantizando una excelente experiencia en smartphones.
* **JavaScript (Vanilla):** Lógica de cálculo en tiempo real y validación de datos del lado del cliente.
* **SEO & Analíticas:** Implementación de metaetiquetas estructuradas, Google Site Verification y métricas de tráfico con Google Analytics (gtag.js).
